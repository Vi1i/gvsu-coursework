These three variables x, y, and line are the backbone on how the data is set up for D3, and ultimately the graph. x and y are how we set up the range of the x and y axis on our graph. Notice how .scaleTime() and .scaleLinear() are different? This is because the x-axis data will be in date format, whereas the y-axis data will be just a typical numerical value. Then, since the x-axis is horizontal, we scale the data to the width of the graph and the y-axis is scaled with the height of the graph. These scale funtions from D3 can be changed to whatever data you might want to use.
